# MandelbrotSet
A program to graph fractal sets developed in C++ using Allegro5. To run in windows: https://www.youtube.com/watch?v=Dz-RbYk3zJo<br>

## Run
```
git clone git@github.com:omarsalazars/MandelbrotSet.git
cd MandelbrotSet
make
./a.out
```
